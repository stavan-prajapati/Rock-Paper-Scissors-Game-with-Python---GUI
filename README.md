# Rock Paper Scissors GUI Game

This is a Rock Paper Scissors game made using Python and Tkinter.
The game runs in a graphical window where the player competes against the computer.

The computer makes a random choice, the result is shown on screen, and scores are updated after every round.
The game can be restarted at any time.

---

REQUIREMENTS

- Python 
- Tkinter
- random module

---

INSTALL REQUIRED MODULES

Install Tkinter using pip:

pip install tkinter

The random module comes built-in with Python, so no installation is needed.

---

FEATURES

- Graphical user interface
- Button based gameplay
- ASCII art display
- Live score tracking
- Restart option
